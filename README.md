## Сборка

Из корневой директории проекта выполнить:

```bash
cmake -S . -B build
cmake --build build
```

После успешной сборки исполняемый файл будет находиться по пути:

```text
build/media_indexer
```

## Запуск

Программа принимает два аргумента командной строки:

```bash
./build/media_indexer <scan_interval_seconds> <directory_path>
```

Где:

- `<directory_path>` — путь к каталогу, который нужно сканировать;
- `<scan_interval_seconds>` — периодичность сканирования в секундах.

Такая последовательность взята опираясь на условие задачи


Пример запуска:

```bash
./build/media_indexer 60 /home/use
```

В этом примере программа будет каждые 60 секунд сканировать каталог `/home/user`.

Программа работает в бесконечном цикле, для остановки используется комбинация Ctrl + C

Так как четкого ТЗ нету, был использован собственный полет фанатазии)
